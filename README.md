## Installation
```bash
git clone https://github.com/nishant4671/pycalc-pro.git
cd pycalc-pro
```

## Usage
### GUI Version (Recommended)
```bash
python gui_calculator.py
```

### CLI Version
```bash
# Basic calculation
python calculator.py "2+3*4"

# Unit conversion
python calculator.py --convert 10 cm inch
```
